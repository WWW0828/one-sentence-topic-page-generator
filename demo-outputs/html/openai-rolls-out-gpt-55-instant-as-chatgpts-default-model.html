Tech / Model Release

GPT-5.5 Instant is now default.

OpenAI made GPT-5.5 Instant the new default ChatGPT model on May 5, 2026, replacing GPT-5.3 Instant across all user tiers. The model delivers 52.5% fewer hallucinated claims on high-stakes prompts, improved math and reasoning benchmarks, more concise answers, and a new auditable memory feature with sources.

Comparison
